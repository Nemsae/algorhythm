JotEasy

ABOUT THE PROJECT

JotEasy is an web app for note taking. Utilizing the Twitter API and websockets we open up a twitter stream set by geolocation. Once the stream is started, the user is able specify by location and radius of the search. Once the stream is stopped we then send that batch of tweets, which can number from a 100 to 10,000, to Watson's Language Alchemy to analyze the tweets. What we specified Watson to return were the entities (subjects of the tweets), concepts, keywords, emotion and sentiment analysis. To make meaning of this data we implemented three graphs displaying the trending hashtags and subjects that people were tweeting about, including the emotional analysis for that cluster of data.

MADE WITH

M.E.R.N. Stack (MongoDB, Express, React, Node), Twitter API, Watson AlchemyLanguage, socket.io, chart.js
